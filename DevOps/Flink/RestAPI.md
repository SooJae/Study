# GET

## /config
설장 정보를 뿌려준다.
```json
{
    "refresh-interval": 3000,
    "timezone-name": "대한민국 표준시",
    "timezone-offset": 32400000,
    "flink-version": "1.9.1",
    "flink-revision": "4d56de8 @ 30.09.2019 @ 11:32:19 CST"
}
```
