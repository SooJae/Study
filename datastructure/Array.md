# Array vs List
a={10,20,30,40}
## a[3] = 50을 추가할 때
Array : 기존의 값을 덮어쓴다. a[3] =50
List : 기존에 있던 값 40을 뒤로 옮긴다. (a[4]=40)
## a[2]을 제거할 때
Array : a[2] = 0 (빈 자리 생성)
List : a[2] = 40 ( 앞으로 당김 )

python : 배열 X, 리스트 O
